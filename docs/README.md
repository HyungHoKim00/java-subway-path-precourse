## 기능 구현 목록

### 입력

- [ ] 경로를 조회할지 종료할지 입력
    - [ ] 조회 시 1, 종료 시 Q 입력, 그 이외의 입력 시 예외 처리
- [ ] 경로 기준 입력
    - [ ] 거리순 시 1, 시간순 시 2, 이전으로 돌아가려면 B
- [ ] 출발역, 도착역
    - [ ] 거리가 없거나, 출발역과 도착역이 동일하거나, 역이 없을 시 예외

### 로직

- [ ] 다익스트라 알고리즘에 따라 최단 출력
    - [ ] 거리순과 시간순 모두 가능하도록 설정

### 출력

- [ ] 메인 화면 출력
- [ ] 기능 목록 출력
- [ ] 조회 결과 출력
